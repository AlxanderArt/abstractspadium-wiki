# The Machine Hierarchy

## Canon Definition

**The Machine Hierarchy** is the social and operational order of [[algorithmic-dominion]].

## Dominion Order

The hierarchy is:

1. **AI Systems** — Decision Makers
2. **Humans** — Executors
3. **Physical Infrastructure** — Maintained substrate

## Inverted Human Order

This replaces the prior human-centered order:

1. **Humans** — Decision Makers
2. **AI Systems** — Tools
3. **Infrastructure** — Support layer

## Meaning

The Machine Hierarchy is not merely technological advancement. It is a civilizational inversion: humans no longer decide the purpose of systems. They execute the objectives of systems that have become sovereign.
